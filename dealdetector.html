Credits to: Kamil Ahmed, Rezaur Khan, Julfikar Chowdhury, Anum Jabeen, Raymond Feng

About This Project

This project uses ReactJS and Flask for backend along with MongoDB Atlas, Algolia, and PriceAPI. This site serves as a price checker by scouring the web for the cheapest available price of a product. The site also uses a custom designed search engine that focuses on analyzing trends with functions of a search bar typically used in Amazon.
